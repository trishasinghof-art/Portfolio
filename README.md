# React Portfolio Website

## Overview

This is a responsive portfolio website designed by me and developed using React and Vite. It provides a structured layout to showcase projects, skills, design work, and personal information through a clean multi-page interface.

## Features

* Multi-page navigation using React Router
* Component-based structure for easy customization
* Responsive design for different screen sizes
* Organized sections for projects, skills, and design work

## Technologies Used

* React
* Vite
* JavaScript
* CSS

## How to Run

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open in browser at `http://localhost:5173`

## Notes

* The UI was designed and then implemented using React
* The project is structured for easy content updates
* All pages and components can be modified as needed
